# GlassGen (Glassmorphism Generator)
GlassGen is a small web tool that allows users to create modern glass-style buttons with a live preview. The interface lets you customize reflection color, opacity, and shadow blur to experiment with different visual effects inspired by glassmorphism design.

As you adjust the controls, the preview updates in real time and generates the corresponding CSS code, making it easy to copy and use in your own projects. The goal of this project is to explore advanced CSS techniques such as gradients, backdrop filters, layered shadows, and interactive UI effects.

This project was built using HTML, CSS, and JavaScript and focuses on clean UI design, responsive layout, and reusable styling patterns. It can be useful for designers and developers who want to experiment with glassmorphism effects or quickly generate CSS for glass-style buttons.
